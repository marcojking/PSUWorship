# PSU Worship — Website UI Kit

A high-fidelity recreation of the PSU Worship public website and internal tools,
lifted from the codebase at `PSUWorship/src/app/` and `src/components/`.

Screens included as click-through prototype:
- **Home** — `/` — dashboard of tool tiles (Harmony, Setlist, Merch, Join)
- **Join the Team** — `/join` — role selection, step progress
- **Setlist Manager** — `/setlist` — song library + setlist dashboard
- **Merch Store** — `/merch` — warm-dark product grid with patchwork hero

Components are small, cosmetic recreations; real data is stubbed.
Open `index.html` to use the kit.
